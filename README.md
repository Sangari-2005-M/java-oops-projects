# java-oops-projects
A collection of Java mini-projects focused on implementing core Object-Oriented Programming (OOP) principles. Each project demonstrates real-world application logic using Encapsulation, Inheritance, and Data Integrity to build robust and maintainable backend systems.


---
## 📝 Projects Included

### 1. Student Report Card System
A Java program that demonstrates **Encapsulation**.
- **Key Concepts:** Data Hiding (Private fields), Getters/Setters, and Business Logic.
- **Functionality:** Calculates total marks and average based on subject inputs.
